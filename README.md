# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview
This is my first data analytics project where I analyze customer shopping behavior using a dataset. The goal is to understand how customers shop, identify patterns, and generate useful insights for decision-making.

🔗 GitHub Repo: https://github.com/aayushchouhan9/Data_analytics_proj

---

## 📂 Dataset Information
- File: `customer_shopping_behavior.csv`
- Contains customer purchase-related data
- Includes information like:
  - Customer details
  - Product categories
  - Purchase behavior
  - Spending patterns

---

## 🎯 Objectives
- Understand customer buying patterns
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Visualize trends using graphs
- Generate insights for business improvement

---

## 🧰 Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights & Conclusion

---

## 📈 Key Insights
- Customers have different shopping patterns based on categories
- Spending behavior varies among users
- Some product categories are more popular than others
- Data can help businesses improve marketing strategies
